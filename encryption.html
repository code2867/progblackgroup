Encryption

This page is about encryption

Encryption is the process of scrambling data using a key into an unreadable format called ciphertext so that, if intercepted, it cannot be understood.

The author of this page is Hannah.


Name Description Pros Cons
Symmetric Encryption The same key is used to both encrypt and decrypt It is easier to set up Far less secure
Asymmetric Encryption A private key is used held by the receiever to decrypt and there is a separate public key used to encrypt More secure and can be used for digital signatures More difficult to implement
