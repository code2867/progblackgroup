How can we use Computer Science to help society?

Game devs

Game devs are responsible for coming up with cool games that not only help people to have fun after a long, busy day, but also provides ways for people to develop better mental health and cope with the daily stresses of life. According to the webmd website, video games can help those dealing with past trauma.

Software developers

Software developers can work in a wide range of different fields to create cool software that helps companies, businesses, and even save lives!
